Welcome to the maven-googlewebtoolkit2-sample

This is an simple example for GWT RPC applications within a Maven Multi-Module project. Specifically, this is a very simple GWT (1.4.61) Client (you are looking at it) that makes a RPC/AJAX server call to a backend Java 1.5 GWT server.
Maven Modules Explained:
  • parent: this is the root/parent of the project
  • war: this is the client GWT code
  • rpc: this is an interface to a GWT server (not an implementation)
  • server: is the implementation of the rpc interface (in Java1.5+)
  • Look inside the pom.xml's for more comments and docs
