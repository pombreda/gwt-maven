maven-googlewebtoolkit-plugin 1.x (gwt-maven) Install

Add the gwt-maven repository location to your maven.repo.remote property

maven.repo.remote=http://mirrors.ibiblio.org/pub/mirrors/maven/,http://gwt-maven.googlecode.com/svn/trunk/mavenrepo/

Download/Install the gwt-maven plugin

maven -DartifactId=maven-googlewebtoolkit-plugin -DgroupId=com.totsp.gwt -Dversion=1.0.3 plugin:download
