maven-googlewebtoolkit-plugin 1.x (gwt-maven)

Overview

The gwt-maven 1.x plugin is used to develop, build and package GWT applications.

gwt-maven can invoke GWTShell and GWTCompiler, merge application deployment resources (web.xml, context.xml) with the GWT Hosted Mode development server instance, and can also easily create deployable WAR files for GWT applications (with the correct GWT RPC service servlet entries automatically injected based on the GWT module file).
